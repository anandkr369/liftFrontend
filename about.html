About Smart Elevator Monitor

Last updated: Just now

Introduction

This project provides a real-time monitoring system for elevators, utilizing IoT technologies like the ESP32 board, cloud computing through Adafruit IO, and a dynamic web interface to display elevator statuses.

Features

  • Real-Time Elevator Data: Fetches floor, door status, and movement information.
  • Request Management: Allows users to request a floor through the dashboard.
  • Cloud Integration: Uses Adafruit IO for cloud data storage and retrieval.
  • Animation and UI: Smooth animations and real-time updates for an interactive experience.

User Manual

Step 1: Connect the ESP32

Connect the ESP32 board to the elevator system and ensure it is connected to a stable Wi-Fi network.

Step 2: Set up Adafruit IO

Set up the Adafruit IO account to fetch and monitor elevator data. Make sure all credentials are stored in the ESP32 device.

Step 3: Use the Dashboard

Access the web dashboard from your browser. The dashboard will display the real-time status of the elevators, including their current floor and door status.

Step 4: Report Issues

If any issues arise, click the "Report Issue" button to notify the admin for further action.
